# Weekly tracking update reference

## Canonical short report template

```markdown
### Trending

<!-- data key="trending" start -->
🟢 on track
<!-- data end -->

### Target date

<!-- data key="target_date" start -->
YYYY-MM-DD
<!-- data end -->

### Update

<!-- data key="update" start -->
[What changed this week, blockers, next actions, ship-point framing]
<!-- data end -->

### Summary

<!-- data key="summary" start -->
[One-paragraph summary]
<!-- data end -->

<!-- data key="isReport" value="true" -->
<!-- data key="howieReportName" value="short" -->
```

## Trend values

- 🟢 on track
- 🟡 at risk
- 🔴 off track
- ⚪ inactive
- ⚪ not planned
- 🟣 done

## Ship-point checklist

- Staff ship: what is required to validate internally?
- Public Preview: what customer-facing scope is ready?
- GA: what remaining quality/readiness work is needed?
